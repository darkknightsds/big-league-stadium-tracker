# Big League Stadium Tracker

##### _Shane Stafford, Android developer_

## Description
This app will allow you to track your favorite MLB stadiums. Add and rank the ones you've been to, make a wishlist for ones you'd like to visit and see what other app users think about your favorite ball parks.
